#  Number plate recognition system for security and crime prevention using Raspberry Pi

## 📌 Overview
This project is a deep learning-based Number Plate Detection system using YOLO.

## 🧠 Features
- Detects number plates from images
- Custom trained dataset
- YOLO model
- TFLite deployment support

## 📁 Project Structure
- src → main code
- models → trained models
- dataset → images and labels
- training → training notebook

## ⚙️ Installation
pip install -r requirements.txt

## ▶️ Run
python src/main.py
